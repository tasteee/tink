z-piano-roll

Draw mode: click-drag on empty grid to place a note. Select mode: drag a marquee to multi-select, double-click empty to place, double-click / right-click a note to delete. Drag a note body to move, grab its left/right ~6px edge to resize. Overlapping a note trims/splits the older one (latest wins). Keys: Del delete · Ctrl/⌘+D duplicate · Ctrl/⌘+A select all · ↑↓←→ nudge (Shift+↑↓ = octave) · Esc clear.
